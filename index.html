Summary: Latitude vs X

This project was aimed to explore the effects of latitude on weather. The original question was: "what happens to the average Temperature as you get closer to the equator? The answer may seem obvious (it gets warmer), but proving it is something different. As such we set out to gather data to support this hypothesis. Using several API requests from weather sites data on temperature, humidity, cloud coverage, and wind speeds were collected for about 500 random cities across the globe. Not surprisingly, as you plot the temperature as a function of latitude, one observes a clear maximum around latitude = 0.
